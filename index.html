Android

Automatic Method:

  1. Click on these links to Download on Android or Huawei Device
  2. After the software has been installed via the Play Store, Download Android Profile
  3. The Profile will redirect you to the Strongswan Application, and will prompt you to enter your username and password
  4. Username: (enter your username)
  5. Password: (enter your password)
  6. Select "SAVE"
Manual Method:
  1. Open the Google Play app.
  2. Search for "Strongswan"
  3. Select "strongSwan VPN Client", and select "Install"
  4. Open the newly installed strongSwan app
  5. Select "ADD VPN PROFILE"
  6. Enter the following information:
  7. Profile Name: VPN Profile
  8. Gateway: vpn.itaniconsulting.com
  9. Type: (Select "IKEv2 EAP (Username/Password")
  10. Username: (enter your username)
  11. Password: (enter your password)
  12. CA certificate: (check mark "Select automatically")
  13. Select "SAVE"
Connecting to the VPN IKEv2 VPN on Android:
  1. Open the strongSwan app
  2. Select the VPN Profile connection
NOTE: You may be prompted to allow strongSwan to intercept network traffic. To continue, select "I trust this application" and select "OK"
You are now connected to the VPN.

Disconnecting from the IKEv2 VPN on Android:
  1. Open the strongSwan app
  2. Select "Disconnect"
You are now disconnected from the IKEv2 VPN.
Note: you can add the quick connect as a widget to connect and disconnect quickly from VPN

iPhone

  1. Download iPhone Profile
  2. Allow the file to be downloaded then close
  3. Go to Settings
  4. Click on the Profile Downloaded
  5. Click on Install
  6. Put ur phone Passcode
  7. Click on Install then Install
  8. Enter you VPN username then next
  9. Enter you VPN password then next
  10. Done
  11. Go to settings and enable VPN with the slider
  12. Congratulations you are now connected to the VPN

Windows

  1. Download Windows App
  2. Allow the file to be downloaded
  3. Open the winprf.exe
  4. Click on yes to install
  5. Click on ok to proceed
  6. Enter you VPN username then next
  7. Enter you VPN password then next
  8. Done, you can now connect.
  9. To Connect again, go to the wifi lan icon in windows
  10. Click on VPN and Connect
  11. Congratulations you are now connected to the VPN
